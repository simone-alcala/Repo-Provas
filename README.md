# Getting started with DRIVEN PASS

## `Backend settings:`
#

## `.env SETUP`

### DATABASE_URL
 - postgres://user:password@host:port/database_name

### PORT=port number
 - Default 5000

### BCRYPT=salts
 - Number of salts, e.g. 10

### JWT_KEY=key
 - Your super secret key

### JWT_EXPIRATION=expiration time
 - Number in miliseconds
 
### MODE=DEV

#
### Run

 - *npm install*
 - *npm run prisma:deploy*
 - *npm run dev*
